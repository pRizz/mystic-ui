## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `pulseColor` | `string` | `"#808080"` | Color of the expanding pulse shadow. |
| `duration` | `string` | `"1.5s"` | Duration of a single pulse cycle. |
| `children` | `JSX.Element` | required | Button content. |
