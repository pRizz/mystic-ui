## Usage

<ComponentUsage
  imports={["AnimatedList"]}
  body={"export default function Example() {\n\treturn (\n\t\t<AnimatedList>\n\t\t\t<p>Item 1</p>\n\t\t\t<p>Item 2</p>\n\t\t\t<p>Item 3</p>\n\t\t</AnimatedList>\n\t);\n}"}
/>

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `delay` | `number` | `1000` | Delay in milliseconds before revealing the next item. |
| `children` | `JSX.Element` | required | List items to reveal. |
