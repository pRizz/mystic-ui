## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `mainCircleSize` | `number` | `210` | Diameter of the innermost ripple circle in pixels. |
| `mainCircleOpacity` | `number` | `0.24` | Starting opacity of the innermost circle. |
| `numCircles` | `number` | `8` | Total number of ripple rings to render. |
| `class` | `string` | `undefined` | Additional classes for the ripple container. |

## Notes

- The ripple layer fills its nearest relatively positioned container.
- Reduce `numCircles` when you want a quieter backdrop behind CTA content.
