## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `shimmerWidth` | `number` | `100` | Width, in pixels, of the animated shine band. |
| `class` | `string` | `undefined` | Additional classes for the paragraph element. |

## Notes

- The component renders a `<p>` by default and also accepts standard paragraph attributes.
- Pair it with muted text colors for the most visible shimmer contrast.
