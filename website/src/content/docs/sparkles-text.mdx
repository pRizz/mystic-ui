title: Sparkles Text
description: A dynamic text that generates continuous sparkles with smooth transitions, perfect for highlighting text with animated stars.
category: text
---

## Usage

<ComponentUsage
  imports={["SparklesText"]}
  body={"export default function Example() {\n\treturn (\n\t\t<SparklesText sparklesCount={12}>A little extra emphasis</SparklesText>\n\t);\n}"}
/>

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `colors` | `string[]` | `["#9E7AFF", "#FE8BBB"]` | Palette used for the generated sparkles. |
| `sparklesCount` | `number` | `10` | Number of sparkle particles rendered around the text. |
| `class` | `string` | `undefined` | Additional classes for the text wrapper. |

## Notes

- Pass your highlighted text as children.
- Increase `sparklesCount` sparingly, since every sparkle is animated independently.
