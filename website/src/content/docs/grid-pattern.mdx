## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `width` | `number` | `40` | Width of each grid cell in the generated pattern. |
| `height` | `number` | `40` | Height of each grid cell in the generated pattern. |
| `x` | `number` | `-1` | Horizontal offset applied to the pattern origin. |
| `y` | `number` | `-1` | Vertical offset applied to the pattern origin. |
| `squares` | `Array<[number, number]>` | `[]` | Grid coordinates to highlight with filled squares. |
| `strokeDasharray` | `string` | `undefined` | Optional SVG dash pattern for the grid lines. |
| `class` | `string` | `undefined` | Additional classes for the wrapping SVG. |

## Notes

- `GridPattern` is absolutely positioned by default, so it works best inside a relatively positioned parent.
- Use `squares` to accent a few cells while keeping the rest of the grid subtle.
