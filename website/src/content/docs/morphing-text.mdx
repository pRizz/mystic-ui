## Usage

<ComponentUsage
  imports={["MorphingText"]}
  body={"export default function Example() {\n\treturn <MorphingText texts={[\"Hello\", \"World\"]} />;\n}"}
/>

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `texts` | `string[]` | required | Array of text values to morph between. |
| `class` | `string` | `undefined` | Additional classes for the wrapper element. |
