## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `speed` | `number` | `1` | Multiplier applied to the gradient background size and perceived animation speed. |
| `colorFrom` | `string` | `"#ffaa40"` | Starting gradient color. |
| `colorTo` | `string` | `"#9c40ff"` | Midpoint and ending gradient color. |
