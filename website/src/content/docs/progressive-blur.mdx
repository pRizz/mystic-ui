---
title: Progressive Blur
description: Add a progressive blur effect to scrollable or layered content.
category: background
---

## Usage

```tsx
import { ProgressiveBlur } from "@/components/mystic-ui/progressive-blur";

export default function Example() {
	return (
		<div class="relative h-64 w-full overflow-hidden">
			<div>{/* your content */}</div>
			<ProgressiveBlur height="50%" position="bottom" />
		</div>
	);
}
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `height` | `string` | `"30%"` | Height of the blur effect when using top or bottom mode. |
| `position` | `"top" \| "bottom" \| "both"` | `"bottom"` | Placement of the progressive blur mask. |
| `blurLevels` | `number[]` | `[0.5, 1, 2, 4, 8, 16, 32, 64]` | Blur strengths used for each stacked blur layer. |
