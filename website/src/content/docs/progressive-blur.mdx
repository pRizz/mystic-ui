## Usage

<ComponentUsage
  imports={["ProgressiveBlur"]}
  body={"export default function Example() {\n\treturn (\n\t\t<div class=\"relative h-64 w-full overflow-hidden\">\n\t\t\t<div>{/* your content */}</div>\n\t\t\t<ProgressiveBlur height=\"50%\" position=\"bottom\" />\n\t\t</div>\n\t);\n}"}
/>

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `height` | `string` | `"30%"` | Height of the blur effect when using top or bottom mode. |
| `position` | `"top" \| "bottom" \| "both"` | `"bottom"` | Placement of the progressive blur mask. |
| `blurLevels` | `number[]` | `[0.5, 1, 2, 4, 8, 16, 32, 64]` | Blur strengths used for each stacked blur layer. |
