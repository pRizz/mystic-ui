## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `number` | `number` | `30` | Total meteors rendered in the effect. |

## Notes

- The effect uses randomly generated positions and timings on mount, so each render feels slightly different.
- Wrap it in a relatively positioned container when you want the meteors clipped to a section.
