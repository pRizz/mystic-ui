## Usage

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `variant` | `"default" \| "outline"` | `"default"` | Visual treatment for the button shell. |
| `size` | `"default" \| "sm" \| "lg" \| "icon"` | `"default"` | Predefined size preset. |
| `children` | `JSX.Element` | required | Button content. |
