## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `duration` | `number` | `20` | Time, in seconds, for one full orbit cycle. |
| `delay` | `number` | `0` | Delay before the orbit animation starts. |
| `radius` | `number` | `50` | Distance from the center point to the orbiting child. |
| `path` | `boolean` | `true` | Whether to render the orbit guide ring. |
| `reverse` | `boolean` | `false` | Reverse the rotation direction. |
| `class` | `string` | `undefined` | Additional classes for the orbit wrapper. |

## Notes

- Render any icon, avatar, or custom node as the child of `OrbitingCircles`.
- The parent container should be positioned relatively so the orbit can center correctly.
