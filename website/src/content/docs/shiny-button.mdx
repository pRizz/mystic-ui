---
title: Shiny Button
description: A shiny button component with a sweeping light effect.
category: component
---

## Usage

```tsx
import { ShinyButton } from "@/components/mystic-ui/shiny-button";

export default function Example() {
	return <ShinyButton>Shiny Button</ShinyButton>;
}
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `children` | `JSX.Element` | required | Button content. |
| `class` | `string` | `undefined` | Additional classes applied to the button wrapper. |
