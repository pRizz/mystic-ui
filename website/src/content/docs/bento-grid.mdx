## Props

### `BentoCard`

| Prop | Type | Description |
| --- | --- | --- |
| `name` | `string` | Primary card heading. |
| `description` | `string` | Supporting card copy. |
| `href` | `string` | Destination for the CTA link. |
| `cta` | `string` | CTA label rendered in the link action. |
| `Icon` | `Component` | Visual icon displayed near the title. |
| `background` | `JSX.Element` | Decorative background content. |
