## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `width` | `number` | `433` | Output width of the SVG device frame. |
| `height` | `number` | `882` | Output height of the SVG device frame. |
| `src` | `string` | `undefined` | Image rendered inside the device screen. |
| `videoSrc` | `string` | `undefined` | Video rendered inside the device screen. |

## Notes

- The component accepts standard SVG attributes in addition to the custom props above.
- Provide either `src` or `videoSrc` depending on whether you want a static screenshot or moving demo.
