## Usage

<ComponentUsage
  imports={["RippleButton"]}
  body={"export default function Example() {\n\treturn <RippleButton>Ripple Button</RippleButton>;\n}"}

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `rippleColor` | `string` | `"#ffffff"` | Color used for the ripple wave. |
| `duration` | `string` | `"600ms"` | Duration of the ripple animation. |
| `children` | `JSX.Element` | required | Button content. |
