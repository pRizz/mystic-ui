---
title: Ripple Button
description: An animated button with a click-triggered ripple effect.
category: component
---

## Usage

```tsx
import { RippleButton } from "@/components/mystic-ui/ripple-button";

export default function Example() {
	return <RippleButton>Ripple Button</RippleButton>;
}
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `rippleColor` | `string` | `"#ffffff"` | Color used for the ripple wave. |
| `duration` | `string` | `"600ms"` | Duration of the ripple animation. |
| `children` | `JSX.Element` | required | Button content. |
