## Usage

<ComponentUsage
  imports={["LetterPullup"]}
  body={"export default function Example() {\n\treturn <LetterPullup text=\"Letter by letter\" />;\n}"}
/>

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `text` | `string` | required | Text split into individual characters for the staggered reveal. |
| `delay` | `number` | `0.05` | Stagger delay between each character animation. |
| `class` | `string` | `undefined` | Additional classes for the wrapper element. |

## Notes

- `LetterPullup` is a Mystic-only text effect.
- Spaces in the input string are preserved so the final layout matches the original copy.
