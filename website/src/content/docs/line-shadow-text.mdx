## Usage

<ComponentUsage
  imports={["LineShadowText"]}
  body={"export default function Example() {\n\treturn <LineShadowText>Magic UI</LineShadowText>;\n}"}

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `children` | `string` | required | Text displayed with the moving shadow effect. |
| `shadowColor` | `string` | `"black"` | Color used for the animated line shadow. |
| `as` | `"article" \| "div" \| "h1" \| "h2" \| "h3" \| "h4" \| "h5" \| "h6" \| "li" \| "p" \| "section" \| "span"` | `"span"` | HTML tag used for the rendered text. |
